Kodun Anlamı

Proc sınıfı bir işlem  için şu bilgileri tutuyor:


id → İşlemin adı veya numarası

arrival → Sisteme geliş zamanı

burst → CPU'da çalışma süresi

remaining → Kalan çalışma süresi 

priority → İşlemin öncelik değeri


start → İşlem ilk kez CPU aldığında zaman

completion → İşlem bittiğinde zaman

waiting → Bekleme süresi

turnaround → çalışma + bekleme toplamı
